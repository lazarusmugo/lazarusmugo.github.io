# Portfolio - Lazarus Mugo

My personal portfolio website built with Next.js, showcasing my work as a Frontend Developer, Mobile Apps Developer, and UI/UX Designer.

## Currently

This portfolio is undergoing a complete rebrand! Check back soon for the updated version.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Font:** Quicksand (Google Fonts)
- **Deployment:** Vercel

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
